# Sorting Station IO Map

## Sensors

| Sensor  | Input AB PLC | Input Siemens PLC | 1-Signal applied at sensor output if: |
|---------|--------------|-------------------|---------------------------------------|
| Part_AV | I:0/0        | I0.0              | Workpiece available                   |
| B2      | I:0/1        | I0.1              | Metallic workpiece                    |
| B3      | I:0/2        | I0.2              | Workpiece not black                   |
| B4      | I:0/3        | I0.3              | Slide Full                            |
| 1B1     | I:0/4        | I0.4              | Branch 1 Retracted                    |
| 1B2     | I:0/5        | I0.5              | Branch 1 Extended                     |
| 2B1     | I:0/6        | I0.6              | Branch 2 Retracted                    |
| 2B2     | I:0/7        | I0.7              | Branch 2 Extended                     |
| S1      | I:0/8        | I1.0              | Start_PB (NO)                         |
| S2      | I:0/9        | I1.1              | Stop_PB (NC)                          |

## Actuators

| Actuator | Output AB PLC | Output Siemens PLC | 1-Signal at actuator input initiates: |
|----------|---------------|--------------------|---------------------------------------|
| K1       | O:0/0         | Q0.0               | Conveyor Motor ON                     |
| 1M1      | O:0/1         | Q0.1               | Branch 1 to Extend                    |
| 2M1      | O:0/2         | Q0.2               | Branch 2 to Extend                    |
| 3M1      | O:0/3         | Q0.3               | Stopper to Retract                    |
| P1       | O:0/8         | Q1.0               | Start_PL                              |
| P2       | O:0/9         | Q1.1               | Stop_PL                               |


Initial Position:
Stopper advanced  
Branch 1 retracted  
Branch 2 retracted  
Conveyor motor off  

Sequence:
Process will start after Start button is pressed. If workpiece is detected using part available (Part_AV) sensor,

Conveyor motor will be on
Color/material identification (make sure you give enough time for the material identification)
If Black workpiece detected, deposit on slide at end of the conveyor

Stopper retracted
Workpiece ejected 
Conveyor motor Off, Stopper to extend
If Silver workpiece detected, deposit on slide at center of conveyor

Branch 2 to extend
Stopper to retract
Workpiece ejected
Conveyor motor Off, Stopper to extend, Branch 2 to retract 
If Red workpiece detected, deposit on slide at start of conveyor

Branch 1 to extend
Stopper to retract
Workpiece ejected
Conveyor motor Off, Stopper to extend, Branch 1 to retract
